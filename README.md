# 🎮 QuizApp — Викторина на Python

Десктопное приложение-викторина с системой регистрации, таблицей рекордов и таймером.

## 🚀 Возможности

- Регистрация и вход пользователей (пароли хешируются)
- Викторина из 10 случайных вопросов (всего 35 вопроса по разным темам)
- Таймер на каждый вопрос
- Подсчёт очков: правильные ответы × 100 − время × 5
- Таблица лучших результатов (ТОП-5)
- Индивидуальный лучший счёт для каждого игрока
- Современный интерфейс на CustomTkinter

## 🛠 Технологии

- Python 3.14+
- CustomTkinter — современный GUI
- SQLite — база данных
- hashlib — хеширование паролей

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/Lurisma/quiz-desktop-app.git

# Перейти в папку проекта
cd quiz-desktop-app

# Установить зависимости
pip install -r requirements.txt

# Добавить вопросы в БД (первый запуск)
python seed_questions.py

# Запустить игру
python main.py
